RvSn00p FAQ

Q: What does the command line argument "tcp:7500|7500||subjecta,subjectb" mean?
A: It is one or more "daemon|service|network|subject" RvSn00p should listen to. You can enter multiple listener definitions on the command line. I use different start scripts to start  RvSn00p in different environments. The "-title" parameter can be used to give the listner session a name.

Q: I cannot start RvSn00p or I get strange errors when starting RvSn00p, what is wrong? 
A: First check that you have the appropriate classpath set for the Java JAR libraries that are required.
     The required libraries are:

  • tibco\tibrv\lib\tibrvj.jar  and
  • tibco\im\java\crimson.jar 
These libraries can be found in the tibco installation directories. Crimson can be found in the Integration Manager installation or at http://xml.apache.org/crimson/

One of the following libraries can also be used to change the text formating of the captured rendezvous message;
  • tibco\rvscript\rvscript..jar
  • tibco\adapter\sdk\java\Maverick4.jar
Maverick4 can be found whithin the TIBCO Adapter SDK installation or within any Tibco Java Adapter, Integration Manager or Message Broker Installation.

Q: My Scrolling wheel on my mouse does not work?
A: If you upgrade your Java runtime to version 1.1.4 the scroll wheel will work.

Q: I cannot copy from the message details pane?
A: If you upgrade your Java runtime to version 1.1.4 the copy will work.

Q: The rvscript functionality does not work?
A: You need to add the rvscript.jar file to your classpath. The version of rvscript needs to be 1.23 or later.

Q: I want functionality added?
