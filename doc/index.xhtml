User’s Guide

Welcome to the User’s Guide for RvSn00p @version@!

RvSn00p is a graphical application that provides developers and operations support personnel with a sophisticated and feature-rich interface for tracing TIBCO Rendezvous™ messages. If you are reading an off-line copy of this document it may be out of date. The RvSn00p SourceForge site always holds the latest copy of this documentation.

RvSn00p allows you to filter out subjects that you are not interested in making it easier to spot important information and messages. Reduce the time required to locate specific messages by using customizable highlight colours.

RvSn00p is cross platform and will run anywhere there is a suitable virtual machine. RvSn00p requires Java 1.4.2 or later.
