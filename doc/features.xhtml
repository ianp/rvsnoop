Feature Summary

RvSn00p provides many features to make your life easier.

Message Ledger (1)

When you startup RvSn00p you find the main interface screen that shows you a ledger like table showing information about all of the captured messages.

Subject Explorer (2)

The subject explorer displays all of the subjects that have been published to in a hierarchical tree-like fashion. You can use this to browse through all of the subscribed subjects.

Filter out the subjects you are not interested in, messages published on any subject element which is not checked will not appear in the ledger.

Hierarchical Tree View (3)

View your messages in a convenient tree format. The tree view can also parse XML messages so you can view their contents easily as well.

Export to HTML or RvScript (4)

Save the current contents of the ledger to a HTML table, this way you can easily include this information in bug reports and training documents.

Save the current contents of the ledger to an rvscript formatted XML file, this is a great way to produce data for testing purposes.

Saved Configurations (5)

You can now choose to save your configuration to a file, this will save all of the subscribers, the state of the subject explorer, and all of the window positions and style settings.

Full Text Searching (6)

Support for full text searches of all messages in the ledger. Search subject names and message contents.
