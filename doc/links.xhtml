Related Links

The descriptions below are based on descriptions taken from the relevant linked pages.

Net Result Consulting
Strategic, integration, and implementation services within the area of telecommunications and datacommunications.
Enterprise Integration Patterns
Dedicated to making the design and implementation of integration solutions easier. The solutions and approaches described here are valid for most integration tools and standards, including TIBCO. Maintained by Gregor Hohpe of ThoughtWorks.
Raccoon
The Raccoon Framework is a framework that has been built above and around the EAI solution chosen for the projects that we have worked with. This project has recently moved from SourceForge, it remains to be seen how active it will be in it's new home.
RvTest Framework
A traditional unit testing framework is not enough for testing messaging systems. RV Test provides a unit testing framework for TIBCO Rendezvous. The project home page says ‘Under Active Development - 30 September 2004’ so I'm not sure how current any of it really is. If anybody from the project reads this, please get in touch with me if it’s still active.
MessageForge
The intent of MessageForge is to make development of applications based on message-oriented middleware easy, transparent, and type-safe. Mentions XML 5 times in the first couple of paragraphs, so all the buzzwords appear to be in place!
Java API for ActiveDatabase
Created during the development of a web-based trading system for a ‘Major Wall Street Bank’. The system used the ActiveDatabase adapter for monitoring databases updates, and using real-time notifications for updating in-memory caches. I'm not sure what real benefit you get from using this framework instead of just subscribing to the bus directly.
Green Hat Viewer
Green Hat Viewer understands Rendezvous traffic, so it can tell the difference between good packets and bad ones. Pretty screenshots and charting capabilities too! Commercially licensed as far as I can tell.
Trading Systems Assiciates
A collection of monitoring and configuration utilities. Commercially licensed.
IT Toolbox's TIBCO Mailing List
A moderated discussion list focusing on TIBCO and EAI issues.
